# SQL Server Catalog Views (SQL Server PDW)
This topic lists the SQL Server catalog views available in SQL Server PDW.  
  
## SQL Server Catalog Views  
These views have a few PDW-specific modifications, such as range differences or possible values, that differ from SQL Server. For more details, click on the links to see the SQL Server PDW documentation.  
  
> [!NOTE]  
> The data types nvarchar(max), uniqueidentifier are supported in SQL Server catalog views only and cannot be used with or for user data. The data types sysname and sql_variant are supported in SQL Server catalog views, in variables, and with the CAST function, but cannot be used in user tables.
